  • 한줄도움말
IP 필터링은 사내에서 접속하거나, 특정 IP로 수집된 데이터를 필터링 할 수 있습니다.
분석에서 제외하고자 하는 IP주소를 등록하면, 해당 IP주소로 방문하는 경우 분석에서 제외됩니다.
IP 필터링 설정은 254자(254byte)까지 저장 가능합니다.
예) 특정 IP 입력: 123.123.23.1 / IP 대역 입력: 123.123.23.1 – 10
